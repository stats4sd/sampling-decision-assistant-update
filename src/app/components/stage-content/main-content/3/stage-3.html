Decisions about the target population

Before you can reach your target population, you need to understand exactly what that population includes. The questions on this stage will build a description that sets out the boundaries and specifications of your target population.

Table 4 “Standard categories of analysis” in the UNHCR Needs Assessment Handbook provides a useful description of criteria for the analytical framework of assessments. This framework will be useful when thinking about the questions below.

In most cases, these populations are constructs that are determined by the objectives for which the study is being set up. Getting the definition of the population right can help reduce bias.
